# setup
- make sure you are on python 3
- `$ pip install -r requirements.txt`
- `$ python manage.py runserver`
- browsable api endpoints are at calendar-share-backend.herokuapp.com/api/

# api endpoints
- /addresses/
- /calendars/
- /events/
- /profiles/
